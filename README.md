# Mahjong Saga — 麻将江湖

一款横版 16:9 麻将主题的浏览器端 Roguelike 割草游戏。玩家扮演骨雕麻将牌「雀」，在雀坛江湖中抵御妖牌围剿，通过凑齐麻将番型激活技能，体验手牌肉鸽 + 割草的战斗循环。

---

## 技术栈

- **原生 HTML5 + CSS3 + ES6**：无框架、无构建工具、零图片资源。
- **渲染**：纯 DOM 绝对定位 + CSS transform 相机偏移，图形全部用 CSS 渐变/阴影/圆角生成。
- **2.5D 骨雕渲染**：麻将牌面、玩家实体、敌人全部使用纯 CSS `box-shadow` 多层夹心工艺。
- **布局**：16:9 横版战场，左侧竹节导航 + 右侧内容画布，底部 14 列 CSS Grid 手牌槽。
- **状态持久化**：`localStorage` 保存元进度与断点存档。
- **模块方式**：所有脚本通过 `<script>` 标签按顺序加载到全局 `window`，无 `import`/`export`。

---

## 快速开始

1. 在项目根目录启动任意静态服务器，例如：
   ```bash
   python -m http.server 8765
   ```
2. 浏览器访问 `http://localhost:8765/s1_save_select.html`。
3. 点击“开启新征程”，在大本营选择英雄与关卡后“轰然出征”。

## 核心玩法

- **点击攻击**：点击敌人造成普攻/暴击，触发吸血、溅射、冰冻等效果。
- **走位**：WASD / 方向键 / 虚拟摇杆 / 点击地面移动。
- **波次与升级**：击杀敌人获得经验与金币，升级时三选一圣物或武器。
- **Boss Lord**：每关最终波次登场，具备弹幕、砸地、狂暴三阶段状态机。
- **时空裂隙 / 无尽深渊**：通关后可选择进入下一循环，敌人属性指数成长。
- **因果继承**：满足特定关卡条件后，下一关获得永久开局加成。

## 局外成长

| 系统 | 说明 |
|------|------|
| **魔王天赋树** | 消耗魔王核心提升生命、速度、吸附、开局双兵等永久天赋。 |
| **英雄酒馆** | 解锁并切换 Knight / Mage / Assassin 三名英雄，各自拥有独特被动。 |
| **皇家锻造** | 锻造武器、护甲、饰品，随机品质与词条（经验、冰冻、迅捷），支持洗练。 |
| **变异保险库** | 结算时有概率解锁负面变异，出征前可选择携带，换取更高核心收益。 |
| **科技树** | 消耗核心强化生命、攻击、暴击等基础属性。 |

## 主要系统

- **武器系统**：6 种神兵（追踪飞刃、环形护体、扇形散射、区域震荡、持续线杀、全屏脉冲），最多 6 槽位（Mage 为 7）。
- **圣物系统**：10 种基础圣物 + 4 种传说超武，提供攻击、暴击、吸血、冰冻、无人机等构筑方向。
- **敌人 AI**：Tanker（正面减伤）、Stalker（三段冲锋）、Shaman（最佳距离 + 图腾 buff）、Boss Lord。
- **战场突变**：每波随机二选一环境突变（引力逆转、狂暴血月等）。
- **怒气 / Overdrive**：击杀与拾取金币积累怒气，满怒后按空格触发 3 秒 Overdrive：冻结全场敌人、武器无 CD、屏幕微震、金色 FCT。
- **套装共鸣**：三件装备出现同一词条时触发光环（3× 迅捷 = 焰痕灼烧，3× 冰冻强化 = 永冻光环）。
- **FCT / 屏幕震动**：伤害/治疗/经验飘字使用 50 节点对象池；暴击、冰冻、Overdrive 拥有独立样式；暴击、Boss 死亡、闪避等触发屏幕震动。

## 数据存档

`localStorage` 中两个关键键：

- `cr_meta.json`：永久进度（核心、天赋、英雄、装备、变异保险库、最高深渊层数）。
- `cr_active_run.json`：当前活跃局（可被战斗页读取以断点续玩）。
