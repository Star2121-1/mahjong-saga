# 🀄 麻将江湖 (Mahjong Saga) - 游戏设计文档 v4.1

## 🌌 一、 世界观

在"雀坛"里江湖中，天地是一张巨大的绿色绒布麻将桌。万物由"万、筒、条、风、箭"五大元炁汇聚而成。近来发生"死牌异变"，未成牌的杂牌化为"妖牌"涌入中原。玩家扮演掌握"绝学胡牌术"的浪人——**"雀"**，手持纯白骨质麻将牌，在妖牌围剿中凑齐"天命番型"，逆天改命。

## 🏠 三、 大本营（江湖茶馆）

### 布局
- 左侧导航栏 (15%)：暗色调竹节造型，固定不动
- 右侧内容画布 (85%)：CSS Grid 自动换行，横向流式布局

### Tab 功能
1. **雀坛招募** — 英雄招募/切换
2. **牌浪洗练** — 天赋养成
3. **百宝牌箩** — 装备检视
4. **血战到底** — 关卡选择
5. **变异保险库** — 携带变异
6. **雀理乾坤** — 成就系统

### Tab 切换
- 3D 翻牌动画 (`rotateY(-90deg)` → `rotateY(0deg)`)
- 严格内存清理：`cancelAnimationFrame` + `clearInterval` 遍历销毁

## ⚔️ 四、 局内战斗

### 布局
- 16:9 横版战场
- 底部 14 列 CSS Grid 天命手牌槽
- 状态条：HP/经验/怒气 (左上角)

### 手牌肉鸽
- 升级 = 摸牌（三选一麻将牌）
- 凑齐顺子 → 技能激活
- 凑齐刻子 → 大招激活
- 胡牌 → Overdrive (10秒役满暴走)

### 实体渲染
- 全部纯 CSS 2.5D 骨雕夹心质感
- 多重 `box-shadow` 压出厚度感
- 零外部图片资产

### 成就系统 (25 个成就)
- 战斗类 (7): 初尝血腥 → 万骨枯、暴击大师、闪避之王、领主猎手、灭世者
- 通关类 (4): 凯旋初征 → 百战不殆 → 传奇雀士、毫发无伤 → 完美无瑕
- 深渊类 (3): 深渊行者 → 无尽深渊 → 虚空之主
- 特殊类 (11): 初次暴走 → 永动机、腰缠万贯 → 富可敌国、极速通关、群雄汇聚、套装共鸣

### 程序化关卡
- 基于深渊层数动态生成
- HP 倍率: 1.15^层, ATK 倍率: 1.12^层
- 刷怪数量: 1.10^层, 间隔递减: 0.05/层
- 额外波次: +1/层
